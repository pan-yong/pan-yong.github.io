## 二维数组

##### 1. 二维数组的定义

> 类型  数组名 [行数] [列数]

```c
int a[10][5];
float scores[5][3];

#define	M	3
#define N	5
int c[M][N]
```

​	二维数组通过下标形式访问，`数组名 [行号] [列号]` ，下标从0开始。

```c
int i, j;
for(i = 0; i < M; i++)
    for(j = 0; j < N; j++)
        scanf("%d", &c[i][j]);  // 第i行第j列数据
```



​	C语言二维数组，采用行优先的方式存储。
