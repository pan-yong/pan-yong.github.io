###### 头文件申明

头文件的引入

```c++
#include<iostream.h>  // 标准库头文件引入
#include "myhead.h"   // 自己头文件引入
```

防卫式申明

```c++
// complex.h
#ifndef __COMPLEX__		// guard  防卫式申明
#define __COMPLEX__

#endif
```
