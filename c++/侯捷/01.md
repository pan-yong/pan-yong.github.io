## C++简介

###### 课程目标

- 培养规范大气的编程习惯。写C++，一出手就是大家。
- 以良好的方式写C++ class  （ 基于对象：Object Based ）
  - class with out pointer members ：complex
  - class with pointer members  : string
- 多个Class之间的关系 （面向对象：Object Oriented）
  - 继承  inheritance
  - 复合  composition
  - 委托  delegation
