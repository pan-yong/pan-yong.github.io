## 相邻比较交换排序（冒泡排序）

1. 下面程序输出结果是（        ）

```c
main()
{
    int a[5] = {3, 4, 2, 5, 1}, i, j, t;
    for (i = 0; i < 2; i++)
        for (j = 0; j < 5-1-i; j++)
            if (a[j] < a[j+1]){
                t = a[j]; a[j] = a[j+1]; a[j+1] = t;
            }
    for (i = 0; i < 5; i++)
        printf("%2d", a[i]);
}
```
