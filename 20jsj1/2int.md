## 整型

```c
#include<stdio.h>
main()
{
    int a = 65;
    printf("%d", a);
}

/* 注释

int  数据类型   整型

65   十进制整型常量    065 八进制整型常量    0x65  十六进制整型常量

%d   十进制整数输出    %o  八进制整数输出    %x    十六进制整数输出 

a    变量名，只能由字母、数字、下划线组成且第一个字母只能是字母或者下划线。

*/
```

课堂练习

```tex

75转十六进制________，转八进制__________

0x75转八进制________，转十进制__________

075转十进制_________，转十六进制________

#include<stdio.h>
main()
{
    int a = 75;
    printf("%x %o\n", a, a);
    
    int b = 0x75;
    printf("%o %d\n", b, b);
    
    int c = 075;
    printf("%d %x\n", c, c);
    
}
```

